# Gerador de Orçamento e Recibos - EKIPHELP

Sistema web para geração de orçamentos e recibos de pagamento profissionais.

## Funcionalidades

- ✅ Criação de orçamentos profissionais
- ✅ Geração de recibos de pagamento
- ✅ Escolha dinâmica entre tipos de documento
- ✅ Exportação em PDF
- ✅ Interface moderna e responsiva
- ✅ Dados da empresa personalizáveis

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- Firebase
- React-to-PDF

## Deploy na Vercel

Esta aplicação está configurada para deploy automático na Vercel:

1. **Fork ou clone este repositório**
2. **Conecte à Vercel:**
   - Acesse [vercel.com](https://vercel.com)
   - Importe o repositório
   - A Vercel detectará automaticamente as configurações
3. **Deploy automático:**
   - Cada push para a branch main fará deploy automático
   - Preview deployments para outras branches

### Configurações da Vercel
- Build Command: `npm run vercel-build`
- Output Directory: `dist`
- Install Command: `npm install`

**Edit a file directly in GitHub**
